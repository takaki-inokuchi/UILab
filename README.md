## 主な機能

- CSS について詳しく説明
- Vercel での公開

## 使用技術

- **フロントエンド**: React, Next.js, TailwindCSS
- **バッチ処理**: Node.js, ts-node
- **CI/CD / 自動化**: GitHub Actions
- **言語**: JavaScript / TypeScript

## 環境構築

### 必要条件

- Node.js 20+
- npm 8+

### 手順

1. リポジトリをクローン
